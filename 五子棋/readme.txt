操作步骤
首先打开服务器端，开启服务器
打开客户端，连接服务器
可以试着打开聊天功能，和下棋功能

规则：服务器设为黑子，黑子先下
	客户端设为白字，白字要等服务器先下之后才能下


两边都点开始之后才能下棋，否则会给出提示，提示自己或者对方还没有下棋

ishei&&!isbai：表示黑棋下了，该白棋了，也是就服务器下了，该客户端了
!ishei&&isbai:表示白棋下了，该黑棋了，也就是客户端下了，该客户端了
这一对的值总是相反，没下一次棋之后，他们都要取反


程序的最初将ishei=false  将isbai=true，这样服务器就可以先下棋了



优点：友情提示很多，会有各种消息提示，对方做了什么，或者自己做了什么，这边都会有提示
缺点：
	代码多，一千多行，看了一下好多重复代码，客户端跟服务器代码几乎一样
	代码乱，只有自己能看懂，什么功能那段代码实现
	界面不够美观
	一些BUG未能发现或者未能处理
